Files in this directory are stored as ".xml.data", not as ".xml", in order to avoid copyright notices to be inserted.
It seems to be simpler approach than removing these notices in the code when samples are to be applied in midPoint GUI.
It is also advisable to keep files as simple as possible: no XML processing instruction, no namespaces.

Names of these files should start with a specification of object type to which these are to be applied.
